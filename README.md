# Francoverse 🌌

Bienvenido a **Francoverse**, mi portafolio web personal donde presento mis proyectos, habilidades y experiencia como desarrollador **Fullstack**. Este proyecto está construido en **React**, **Next.js**, **Tailwind CSS**, y **Three.js**, presenta animaciones y efectos visuales, incluyendo un fondo de partículas interactivo.

---

## 🎯 Objetivo del proyecto

Francoverse busca:

- Mostrar mis proyectos y habilidades de manera interactiva y atractiva.
- Demostrar mis conocimientos en desarrollo frontend y backend.
- Proveer una experiencia de usuario fluida, responsive y moderna.
- Explorar efectos visuales avanzados con Three.js y React Three Fiber.

---

## 🛠 Tecnologías utilizadas

- **Frontend:** Next.js, React, Tailwind CSS, Framer Motion, React Icons.
- **Visual Effects:** Three.js, React Three Fiber, tsparticles.
- **Backend / API:** Node.js, Express (si aplicara a futuros proyectos).
- **Despliegue:** Azure App Service.
- **Control de versiones:** Git y GitHub.

---

## 📌 Características principales

- Hero Section interactivo con animaciones y partículas de fondo.
- Secciones de habilidades, proyectos y contacto.
- Diseño **responsive** que funciona correctamente en móviles y tablets.
- Componentes reutilizables y modularizados.
- Animaciones fluidas con Framer Motion.
- Integración de partículas y efectos visuales con React Three Fiber y tsparticles.

---

## 🚀 Instalación y uso

Clona este repositorio:

```bash
git clone https://github.com/francovillagra/spaceportfolio.git
cd spaceportfolio
```

## Instala dependencias:
```
yarn install
# o npm install
```
## Inicia el servidor de desarrollo:
```
yarn dev
# o npm run dev
```
Abrir el navegador en http://localhost:3000 para ver la web en funcionamiento.

📂 Estructura del proyecto
```
app/             # Directorio principal de Next.js 13+ (rutas y páginas)
components/      # Componentes UI y secciones
lib/             # Librerías y hooks personalizados
utils/           # Funciones y utilidades
public/          # Recursos estáticos (imágenes, iconos, videos)
constants/       # Constantes globales del proyecto
styles/          # Archivos CSS y Tailwind
```
